    ##String

    字符串是极其常用的重要工具之一。

    性能陷阱

    String不可变，除非编译器做常量优化，否则连接字符串实际是在不断生成新的String对象——耗费内存

    这也是StringBuilder和StringBuffer存在的意义。

    因此，除非确定字符串能被编译器优化掉，都应该使用StringBuilder或者StringBuffer来生成字符串。
